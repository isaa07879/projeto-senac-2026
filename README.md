# viajei

o objetivo do projeto *Viajei* é, compartilhar através das experiencias que vivemos,para que outras pessoas tambem possam usufruir da mesma experiencia. Essas experiencias podem ser sobre qualquer tema(livros,series,cozinha,esportes,viagens...)
   o *Viajei* ,por razoes apropriadas, é indicado para maiores de 14(quatorze) anos.
   ## o que é?

   o viajei sera uma plataforma digital, em que as pessoas podem compartilhar suas experiencias em formato de texto e deixar suas recomendação para outras pessoas
